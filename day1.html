Day 1: Leg/Glute Day Options

Workout Option 1

This workout will hit both legs and glutes with weights but will target the glutes more. Add weight every week to challenge yourself.

  • Sumo Squats w/ Dumbell 3 sets x 12 reps
  • Curtsy Lunge with Dumbell on shoulders 3 sets X 10 each leg
  • Hip Abduction Machine (Push out) 3 sets x 12 reps
  • Hip Adbuction Machine (Squeeze legs together) 3 sets x 12 reps
  • Cable Kickbacks 3 sets x 12 each leg
More Glute Workouts
Workout Option 2

This one still makes use of weights but is more focused on the quads and hamstrings.

  • Leg Extension Machine 3 sets x 12 reps
  • Barbell Squats 3 sets x 12 reps
  • Hamstring Leg Curl Machine 3 sets x 12 reps
  • Calf raises w/ weight 20 regular, 20 w/ toes pointed out, 20 w/ toes pointed in
  • Lunges 3 sets x 10 each leg
More Leg Workouts
Workout Option 3

This workout is meant to make use of resistance bands and be an at home option.

  • Single Leg raises 3 sets x 20 each leg
  • Clamshells 3 sets x 20 each leg
  • Hip Raises 3 sets X 30 reps
  • Squat Jumps 3 sets x 15 jumps
  • Donkey Kickbacks 3 sets x 15 each leg

Tips

  • To warm up go for the stairmaster for about 5-10 minutes.
  • Be sure to stretch
